# CySent 2-Minute Pitch

## Problem
Security teams need fast, consistent response decisions under evolving attack pressure.

## Solution
CySent is an autonomous cyber defense command center where:
- RED simulates adaptive attackers.
- BLUE uses PPO as a stable default defender.
- Hugging Face LLM can be enabled for strategic assist with PPO fallback.

## Why it matters
- Decision speed: automated turn-by-turn actions.
- Explainability: events, narrative timeline, replay export.
- Evidence: benchmark suite with JSON/CSV outputs.

## Product readiness
- FastAPI backend + Next.js frontend.
- Docker and docker-compose included.
- HF Spaces launch files included.
- Environment template and one-command run scripts included.

## Ask
Pilot CySent for SOC training, policy stress testing, and autonomous playbook exploration.
